# 🚀 Sobre mí

Soy un Ingeniero de Software apasionado por crear soluciones que resuelvan problemas reales.

Me especializo en el desarrollo de aplicaciones backend, arquitecturas modernas, automatización de infraestructura y despliegues en la nube.

Disfruto diseñando sistemas que sean rápidos, seguros y preparados para crecer.

Siempre estoy aprendiendo nuevas tecnologías, explorando mejores prácticas y buscando formas de construir software de mayor calidad.

# 💡 Mi filosofía

No creo que un buen desarrollador sea quien conoce más tecnologías.

Creo que un buen ingeniero es quien logra construir software que otros puedan entender, mantener y mejorar.

Por eso siempre intento desarrollar soluciones que sean:

- Simples
- Escalables
- Automatizadas
- Seguras
- Mantenibles
- Orientadas a resultados
